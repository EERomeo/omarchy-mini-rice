For the full functionality you need the included scripts.

**clear-caches.sh** will clear memory caches when you middle click over the mem in the waybar.

**mediaplayer.py** gives you control over what's currently palaying in the waybar middle.
From: https://github.com/Alexays/Waybar/wiki/Module:-Custom:-Examples

**privacydots.sh** gives you witness dots for camera/microfone/location useage.
From: https://github.com/alibaghernejad/waybar-privacy-dots

**rand-wallpaper.sh** will cycle your theme backgrounds every 5 minutes.

omarchy.ttf and omarchy.woff should be placed in ./local/share/fonts replacing existing for the Omarchy logo.
